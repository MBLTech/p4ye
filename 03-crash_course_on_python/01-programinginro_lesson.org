* Programming

** Program

'A list of Instructions that the computer has to follow to reach a goal'

** Syntax

'The set of rules, how a sentence is constructed'
In programming terms,
'The rules for how each instruction is written'

** Semantics

'The actual meaning of a Syntax'
In programming terms,
'The effect the instructions have'

** Script

'The code with short development cycle and can deploy rapidly'

* Automation

'The process of replacing manual step with one that happens automatically'
